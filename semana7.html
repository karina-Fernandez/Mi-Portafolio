SEMANA 7

Uso de Hooks en React

Hooks avanzados de React para gestión de estado, efectos, optimización de rendimiento, y creación de custom hooks reutilizables para lógica compleja.

Temas Aprendidos

Hooks Fundamentales - useState, useEffect, useContext

Los hooks fundamentales forman la base de la programación funcional en React. useState maneja estado local con actualizaciones inmutables, useEffect gestiona efectos secundarios como API calls, subscripciones, y cleanup, mientras useContext proporciona acceso a datos globales sin prop drilling. Dominé patrones como lazy initial state, functional updates, dependency arrays optimizadas, y context providers con valores memoizados para evitar re-renders innecesarios.

Hooks de Optimización - useMemo, useCallback, React.memo

Los hooks de optimización previenen cálculos costosos y re-renders innecesarios. useMemo memoiza valores computados, useCallback memoiza funciones para evitar recreación en cada render, y React.memo optimiza componentes comparando props. Aprendí cuándo y cómo usar estas optimizaciones efectivamente, evitando over-optimization, y técnicas de profiling con React DevTools para identificar bottlenecks de rendimiento en aplicaciones complejas.

Hooks Avanzados - useReducer, useRef, useImperativeHandle

useReducer maneja estado complejo con lógica de actualización centralizada, ideal para state machines y acciones múltiples. useRef accede a elementos DOM directamente y persiste valores mutables entre renders sin causar re-renders. useImperativeHandle personaliza la instancia expuesta por refs en componentes forwardRef. Implementé patrones como reducer composition, ref forwarding, y imperative APIs para casos de uso específicos como focus management y animaciones.

Custom Hooks - Reutilización de Lógica Stateful

Los custom hooks encapsulan lógica stateful reutilizable entre componentes. Creé hooks para casos comunes como useLocalStorage, useFetch, useDebounce, useIntersectionObserver, y useKeyboard. Aprendí patrones de diseño para hooks composables, manejo de cleanup automático, error boundaries en hooks, y testing strategies. Los custom hooks promueven separation of concerns y facilitan testing unitario de lógica compleja.

Ejercicios de Laboratorio - Resultados

Ejercicio 01

State management con useReducer

Objetivo: Implementar gestión de estado complejo con useReducer.

Desarrollé una aplicación de carrito de compras usando useReducer para manejar acciones complejas: agregar/remover items, actualizar cantidades, aplicar descuentos, y calcular totales. Implementé middleware para logging de acciones, undo/redo functionality, y persistencia automática en localStorage. Creé reducers composables para diferentes secciones del estado y testing exhaustivo de cada action type.

Ejercicio 02

Optimización de rendimiento avanzada

Objetivo: Optimizar una aplicación con miles de elementos.

Optimicé una aplicación de tabla con 10,000+ filas implementando virtualización con react-window, memoización estratégica con useMemo/useCallback, y React.memo con comparadores personalizados. Agregué debouncing para búsqueda, lazy loading de datos, y code splitting con React.lazy. Utilicé React DevTools Profiler para identificar bottlenecks y logré reducir el tiempo de render en 80%.

Ejercicio 03

Custom hooks library

Objetivo: Crear una librería de custom hooks reutilizables.

Desarrollé una librería completa de custom hooks: useApi para llamadas HTTP con cache y retry, useForm para manejo de formularios con validación, useLocalStorage con sincronización entre tabs, useIntersectionObserver para lazy loading, useKeyboard para shortcuts, y useTheme para dark mode. Cada hook incluye TypeScript types, documentación completa, y tests unitarios con React Testing Library.

Ejercicio 04

Real-time chat con hooks avanzados

Objetivo: Construir aplicación real-time con WebSockets.

Creé una aplicación de chat en tiempo real usando WebSockets con custom hooks para conexión (useWebSocket), gestión de mensajes (useMessages), y presencia de usuarios (usePresence). Implementé reconexión automática, queue de mensajes offline, typing indicators, y notificaciones push. Utilicé useReducer para estado complejo del chat y Context API para compartir estado entre componentes.

Ejercicio 05

Editor de código con hooks complejos

Objetivo: Desarrollar un editor de código con funcionalidades avanzadas.

Construí un editor de código completo con syntax highlighting, autocompletado, y múltiples tabs. Implementé custom hooks para manejo de archivos (useFileSystem), historial de cambios (useHistory), búsqueda y reemplazo (useSearch), y configuración de editor (useEditorConfig). Agregué colaboración en tiempo real, vim keybindings, y extensiones mediante plugin system. Utilicé useImperativeHandle para APIs imperativas y optimización extrema para manejar archivos grandes.

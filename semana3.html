SEMANA 3

HTML Básico y Avanzado, CSS Básico y Avanzado

Fundamentos y técnicas avanzadas de HTML para estructura semántica y CSS para diseño, layout y presentación visual en el desarrollo web moderno.

Temas Aprendidos

HTML Básico - Estructura y Semántica

HTML (HyperText Markup Language) es el lenguaje de marcado estándar para crear páginas web. Aprendí la estructura básica con elementos como <html>, <head>, <body>, etiquetas semánticas como <header>, <nav>, <main>, <section>, <article>, <footer>, y elementos de contenido como párrafos, listas, enlaces, imágenes y formularios. La semántica correcta mejora la accesibilidad y SEO.

HTML Avanzado - Formularios y Multimedia

Exploré elementos avanzados como formularios complejos con validación HTML5, tipos de input especializados (email, date, range, color), elementos multimedia (<video>, <audio>, <canvas>), APIs modernas como Web Storage, Geolocation, y técnicas de optimización como lazy loading de imágenes, meta tags para SEO, y structured data para rich snippets.

CSS Básico - Estilos y Selectores

CSS (Cascading Style Sheets) controla la presentación visual de HTML. Dominé selectores básicos (elemento, clase, ID), propiedades fundamentales (color, font, margin, padding, border), el modelo de caja (box model), unidades de medida (px, em, rem, %, vh, vw), y conceptos de cascada, especificidad y herencia que determinan cómo se aplican los estilos.

CSS Avanzado - Layout y Animaciones

Implementé técnicas avanzadas como Flexbox para layouts unidimensionales, CSS Grid para layouts bidimensionales complejos, posicionamiento avanzado (relative, absolute, fixed, sticky), transformaciones 2D y 3D, animaciones con @keyframes, transiciones suaves, variables CSS (custom properties), y técnicas de responsive design con media queries.

Ejercicios de Laboratorio - Resultados

Ejercicio 01

Estructura semántica HTML5

Objetivo: Crear páginas web con estructura semántica correcta.

Desarrollé páginas web utilizando elementos semánticos HTML5 apropiados: <header> para encabezados, <nav> para navegación, <main> para contenido principal, <section> para secciones temáticas, <article> para contenido independiente, <aside> para contenido relacionado, y <footer> para pie de página. Implementé jerarquía correcta de headings (h1-h6) y atributos de accesibilidad.

Ejercicio 02

Formularios avanzados con validación

Objetivo: Crear formularios complejos con validación HTML5.

Construí formularios complejos utilizando diversos tipos de input (email, tel, date, range, color), implementé validación nativa HTML5 con atributos como required, pattern, min, max, y minlength. Agregué elementos como <datalist> para autocompletado, <fieldset> y <legend> para agrupación, y personalicé mensajes de validación con JavaScript.

Ejercicio 03

Layouts responsivos con Flexbox

Objetivo: Implementar layouts flexibles y responsivos.

Creé layouts responsivos utilizando Flexbox con propiedades como display: flex, flex-direction, justify-content, align-items, flex-wrap, y flex-grow/shrink/basis. Implementé navegaciones horizontales y verticales, centrado perfecto de elementos, distribución equitativa de espacio, y layouts que se adaptan automáticamente a diferentes tamaños de pantalla.

Ejercicio 04

CSS Grid para layouts complejos

Objetivo: Crear layouts bidimensionales con CSS Grid.

Desarrollé layouts complejos usando CSS Grid con propiedades como grid-template-columns/rows, grid-gap, grid-area, y funciones como repeat(), minmax(), fr unit. Creé layouts de magazine, dashboards, y galerías de imágenes que se reorganizan automáticamente. Implementé áreas de grid nombradas para mayor claridad y mantenibilidad del código.

Ejercicio 05

Animaciones y transiciones CSS

Objetivo: Implementar animaciones fluidas y efectos visuales.

Creé animaciones complejas usando @keyframes para animaciones de carga, hover effects, y micro-interacciones. Implementé transiciones suaves con transition property para cambios de estado, transformaciones 2D y 3D con transform (rotate, scale, translate), y efectos avanzados como parallax scrolling y animaciones activadas por scroll usando Intersection Observer API.
